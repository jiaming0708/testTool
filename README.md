# 說明
這是Angular線上讀書會第三季第二堂的demo

介紹`Jasmine` & `Protractor`，可搭配[講義](https://jiaming0708.github.io/2017/10/17/jasmine-protractor/)

# 使用
執行jasmine測試
> npm test

執行protractor測試前，需先安裝
> npm install -g protractor

測試protractor需要執行兩個指令
> npm run server
> npm run e2e
